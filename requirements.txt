ARM64 汇编测试环境工具链需求 (Requirements)
==============================================================================

本测试程序必须在 macOS 环境下运行。为了能够编译、链接和执行 ARM64 汇编，

1. macOS 命令行开发者工具 (Command Line Tools)
   --------------------------------------------------------------------------
   * 包含组件: Apple Clang (编译器), as (汇编器), ld (链接器)
   * 用途: 将生成的 ARM64 汇编源码 (.s) 编译并链接为可执行文件
   * 检查命令: 打开终端输入 `clang --version`
   * 安装命令: 如果未安装，请在终端输入以下命令，并在弹出的官方窗口中点击“安装”：
     xcode-select --install

2. Python 3 运行环境
   --------------------------------------------------------------------------
   * 版本要求: Python 3.8 或以上版本
   * 用途: 运行自动化测试引导和评估脚本
   * 检查命令: 打开终端输入 `python3 --version`
